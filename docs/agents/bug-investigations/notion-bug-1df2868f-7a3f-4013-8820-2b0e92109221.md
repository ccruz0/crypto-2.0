## Bug details

- **Reported symptom**: Mensaje observado:

AUTOMATIC ORDER CREATION FAILED
Symbol: BTC_USD
Side: BUY
Signal: BUY signal detected
Trade enabled: True
Error: CONFIGURACIÓN REQUERIDA
El campo “Amount USD” no está configurado para BTC_USD.

Problemas detectados:
1. La alerta se repite continuamente.
2. El sistema no está respetando las reglas definidas para las alertas.
3. El sistema no parece aplicar correctamente la lógica de deduplicación o control de spam de alertas.
4. No está claro si el problema es de configuración del dashboard o de validación en el backend.

Objetivos:
1. Revisar toda la documentación del sistema de alertas.
2. Revisar el código del backend que genera alertas y crea órdenes automáticas.
3. Comparar las reglas documentadas vs. lo que realmente hace el código.
4. Detectar discrepancias entre: documentación, implementación del backend, configuración del dashboard.
5. Identificar por qué las alertas se repiten en lugar de enviarse solo una vez o con throttling.

Aspectos concretos a revisar:
- Lógica de deduplicación de alertas
- Control de frecuencia de alertas
- Manejo de errores de configuración (Amount USD missing)
- Validaciones previas antes de enviar alertas
- Reglas definidas en la documentación de alertas
- Integración: signal detection → order creation → alerting
- Sistema de estado para evitar alertas repetidas

Resultados esperados:
1. Lista clara de inconsistencias entre documentación y código.
2. Identificación exacta del módulo o archivo donde se genera el comportamiento incorrecto.
3. Explicación de por qué las alertas se repiten.
4. Propuesta de mejora de arquitectura si es necesario.

Entrega final requerida:
1. Informe técnico (causas, discrepancias, impacto).
2. Propuesta de solución.
3. Prompt para Cursor que implemente la solución.

Antes de generar el prompt para Cursor:
- Simular el comportamiento con la solución propuesta para confirmar:
	- La alerta no se repite.
	- Se respetan las reglas de alertas.
	- Los errores de configuración se manejan correctamente.
